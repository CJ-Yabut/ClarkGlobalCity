# Clark Global City – Static Web Page
**Final Project for Reading in Philippine History**

## Overview
This project is a static web page that explores the historical, cultural, and urban transformation of **Clark Global City** in Pampanga, Philippines. The website showcases how the area evolved from indigenous Kapampangan land, to a U.S. military air base, through the devastation of Mount Pinatubo, and finally into a modern global business hub.

The project emphasizes not only urban development but also the region's cultural heritage, highlighting the resilience and identity of the local communities.

## Features
- **Introduction**: Presents the purpose and significance of studying Clark Global City.
- **History**: Chronological development from pre-colonial times to present day.
- **Interactive Timeline**: Key milestones of Clark Global City's transformation.
- **Cultural Identity**: Highlights Kapampangan heritage and the impact of modernization.
- **Reflection**: Concluding insights about balancing progress with cultural preservation.
- **Responsive Design**: Mobile-friendly and visually organized for ease of reading.
